## Comandos para Utilização desta Aplicação Frontend

- Instale o [NodeJs](https://nodejs.org/pt-br/download/)

- Instale as dependências: `npm i`

- Inicie o servidor web: `npm start`

- Visualize a aplicação: `localhost:3000`
